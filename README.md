# 1C:RSV — Лендинг-страница

Презентационный лендинг для плагина **1C:RSV MCP Server**.

## Содержимое папки

```
landing/
  index.html        ← основной файл лендинга (весь CSS и JS встроен)
  images/           ← изображения
    main-screen.png  ← главный экран
    infographic.png  ← инфографика архитектуры
    install.jpg      ← скриншот установки
    statusbar.jpg    ← строка состояния
    led.jpg          ← LED-индикатор
    port.jpg         ← настройки порта
  README.md         ← этот файл
```

## Способ 1: GitHub Pages (рекомендуется)

### Шаг 1 — Публикация на GitHub Pages

1. Убедитесь что папка `landing/` закоммичена в ваш репозиторий
2. Зайдите в репозиторий на GitHub → **Settings → Pages**
3. В разделе **Source** выберите:
   - Branch: `master` (или `main`)
   - Folder: `/ (root)` — если хотите чтобы лендинг был корнем,
     ИЛИ настройте кастомный путь
4. Нажмите **Save**
5. Через 1-2 минуты сайт будет доступен по адресу:
   ```
   https://ВАШ-ЛОГИН.github.io/ВАШ-РЕПОЗИТОРИЙ/landing/
   ```

> **Совет:** Можно создать отдельный репозиторий `edt-ai-1c.github.io`
> и положить туда только содержимое папки `landing/` — тогда лендинг
> будет доступен по красивому адресу `https://edt-ai-1c.github.io`

### Шаг 2 — Встраивание в Google Sites через iframe

1. Откройте ваш Google Sites
2. Нажмите **Вставить → Вставить HTML** (кнопка `<>`)
3. Вставьте код:
   ```html
   <iframe
     src="https://ВАШ-ЛОГИН.github.io/ВАШ-РЕПОЗИТОРИЙ/landing/"
     width="100%"
     height="900"
     frameborder="0"
     style="border:none;"
   ></iframe>
   ```
4. Нажмите **Следующий** → настройте размер → **Вставить**

> **Замечание:** Google Sites ограничивает высоту iframe — рекомендуется
> установить высоту 900px и позволить пользователям прокручивать iframe,
> либо использовать лендинг как самостоятельную страницу.

## Способ 2: Прямой HTML в Google Sites

> ⚠ Google Sites бесплатная версия ограничивает встроенный HTML.
> Рекомендуется Способ 1 через GitHub Pages.

1. Скопируйте содержимое `index.html`
2. В Google Sites → **Вставить → Вставить HTML**
3. Вставьте HTML-код

**Ограничения бесплатной версии Google Sites:**
- Максимальный размер встроенного HTML: ~1 МБ
- Некоторые CSS-анимации могут быть ограничены из-за Content Security Policy

## Редактирование лендинга

Все тексты, цвета и изображения задаются в `index.html`.

### Быстрое изменение контактов
Найдите в файле `https://t.me/edt_ai_1c` — это ссылка на Telegram-канал.

### Изменение цветов
В начале CSS найдите секцию `:root { }` и измените переменные:
- `--blue: #1d4ed8` — основной синий цвет
- `--indigo: #6366f1` — фиолетовый акцент

### Добавление изображений
Поместите изображения в папку `images/` и обновите пути в `index.html`.
