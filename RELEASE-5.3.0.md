## Код-ревью BSL

### Автоматическая проверка кода по стандартам 1С

Появился новый инструмент `code_review` — автоматическое ревью кода BSL по стандартам разработки 1С: сложность методов, «дурно пахнущий» код, нарушения соглашений (магические числа, слишком длинные строки, повторяющиеся литералы, неправильный тип общего модуля и многое другое). Это проверки, которых нет в обычной проверке EDT, — они её дополняют, а не заменяют.

Работает так: агент запускает код-ревью и получает список замечаний — каждое с точным номером строки, понятным текстом и ссылкой на описание правила. Дальше агент сразу правит нужные строки. Проверить можно весь проект целиком, один модуль или сразу пачку модулей — это удобно, когда вы с агентом только что правили несколько модулей и хотите проверить именно их.

Какие проверки выполнять — настраивается галочками прямо в EDT (Параметры → MCP:RSV → Код-ревью). Снятые галочки теперь учитываются и при вызове агентом, так что ручная проверка из EDT и проверка через агента работают по одному набору правил.

Инструмент работает на бесплатном открытом компоненте «MCP:RSV Code Review» (на базе BSL Language Server). Его нужно один раз установить, дальше он скачивается по постоянной ссылке: https://github.com/prepod2003/mcp-rsv-codereview/releases/latest/download/MCP-RSV-CodeReview.zip
