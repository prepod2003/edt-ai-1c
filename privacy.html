Политика конфиденциальности

Обработка персональных данных при использовании сервиса 1C:RSV MCP Server

1. Общие положения

1.1. Настоящая Политика определяет порядок обработки персональных данных пользователей.

1.2. Оператором персональных данных является ИП Радзивиллович Сергей Владиславович.

1.3. Политика разработана в соответствии с Федеральным законом №152-ФЗ «О персональных данных».

1.4. Согласие на обработку персональных данных предоставляется путем установки галочки при оформлении заказа и акцепте оферты.

2. Какие данные собираются

Оператор собирает только те данные, которые необходимы для исполнения лицензионного договора:

Данные Цель сбора Основание
Адрес электронной почты Отправка кода активации лицензии Исполнение договора (ст. 6 ч. 1 п. 5 ФЗ-152)
Machine ID компьютера Привязка лицензии к конкретному устройству Исполнение договора

Также могут собираться:

  • IP-адрес и технические данные браузера (при использовании сайта)

Оператор не собирает: ФИО, номера телефонов, паспортные данные, данные о местоположении, данные банковских карт (платёж обрабатывается платёжной системой Т-Банка напрямую).

3. Цели обработки персональных данных

  • Исполнение лицензионного договора: отправка кода активации на email после оплаты
  • Техническая поддержка: ответы на вопросы пользователя по его обращению
  • Управление лицензией: переактивация при смене оборудования

Персональные данные не используются в маркетинговых целях и не передаются третьим лицам для рекламы.

4. Хранение данных и защита данных

4.1. Персональные данные обрабатываются в автоматизированных информационных системах Оператора, включая использование облачных сервисов.

4.2. Срок хранения:

  • срок действия лицензии
  • + 1 год после

По истечении срока данные уничтожаются с составлением акта.

4.3. Оператор принимает организационные и технические меры для защиты персональных данных от несанкционированного доступа.

5. Передача данных

Данные передаются:

  • Т-Банк
  • ОФД / касса

Передача осуществляется только в объеме, необходимом для исполнения договора.

В случае использования облачных сервисов может осуществляться трансграничная передача персональных данных с соблюдением требований законодательства РФ.

6. Cookies

Сайт может использовать cookies.

Пользователь вправе:

  • получить информацию о своих персональных данных
  • требовать уточнения данных
  • требовать блокировки или удаления
  • отозвать согласие на обработку
  • ограничить обработку данных

Для реализации прав обратитесь по email: prepod2003@yandex.ru или через Telegram: @Prepod2003.

8. Изменение Политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия публикуется на сайте. Продолжение использования сервиса после публикации означает согласие с обновлённой Политикой.
