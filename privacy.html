Политика конфиденциальности

Обработка персональных данных при использовании сервиса 1C:RSV MCP Server

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки персональных данных пользователей сервиса 1C:RSV MCP Server.

1.2. Оператором персональных данных является ИП Радзивиллович Сергей Владиславович (ОГРНИП 304784735800264, ИНН 780219382520, г. Санкт-Петербург).

1.3. Политика разработана в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных».

1.4. Используя сервис и принимая условия оферты, Пользователь даёт согласие на обработку своих персональных данных в соответствии с настоящей Политикой.

2. Какие данные собираются

Оператор собирает только те данные, которые необходимы для исполнения лицензионного договора:

Данные Цель сбора Основание
Адрес электронной почты Отправка кода активации лицензии Исполнение договора (ст. 6 ч. 1 п. 5 ФЗ-152)
Machine ID компьютера Привязка лицензии к конкретному устройству Исполнение договора

Оператор не собирает: ФИО, номера телефонов, паспортные данные, данные о местоположении, данные банковских карт (платёж обрабатывается платёжной системой Т-Банка напрямую).

3. Цели обработки персональных данных

  • Исполнение лицензионного договора: отправка кода активации на email после оплаты
  • Техническая поддержка: ответы на вопросы пользователя по его обращению
  • Управление лицензией: переактивация при смене оборудования

Персональные данные не используются в маркетинговых целях и не передаются третьим лицам для рекламы.

4. Хранение и защита данных

4.1. Email-адрес хранится в базе данных оператора (Google Sheets) в течение срока действия лицензии и до 1 года после её истечения — для целей возможной переактивации и технической поддержки.

4.2. После истечения указанного срока данные удаляются.

4.3. Оператор принимает организационные и технические меры для защиты персональных данных от несанкционированного доступа.

5. Передача данных третьим лицам

Персональные данные (email) могут быть переданы следующим лицам исключительно в целях исполнения договора:

  • Т-Банк (АО «Тинькофф Банк») — для проведения платежа (email передаётся в чек согласно требованиям 54-ФЗ)

Передача данных в иных целях или иным лицам не осуществляется.

6. Права субъекта персональных данных

Пользователь вправе:

  • Запросить информацию об обрабатываемых персональных данных
  • Потребовать уточнения, блокировки или уничтожения данных
  • Отозвать согласие на обработку персональных данных

Для реализации прав обратитесь по email: prepod2003@yandex.ru или через Telegram: @Prepod2003.

Важно: отзыв согласия на обработку персональных данных означает невозможность исполнения лицензионного договора (не будет возможности отправить код активации).

7. Изменение Политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия публикуется на сайте. Продолжение использования сервиса после публикации означает согласие с обновлённой Политикой.
