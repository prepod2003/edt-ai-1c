Публичная оферта на предоставление лицензии на программное обеспечение 1C:RSV MCP Server

1.1. Настоящее Соглашение является публичной офертой в соответствии со статьёй 437 Гражданского кодекса РФ и регулирует условия предоставления лицензии на ПО 1C:RSV MCP Server.

1.4. Используя ПО, Пользователь подтверждает, что ознакомился с настоящим Соглашением и принимает его условия.

2. Предмет соглашения

2.1. Исполнитель предоставляет Пользователю право использования ПО 1C:RSV MCP Server на условиях простой (неисключительной) лицензии согласно статье 1235 ГК РФ на срок выбранного тарифа.

1C:RSV MCP Server — плагин для 1C:Enterprise Development Tools (1C:EDT) версии 2025.2.1 и выше, обеспечивающий взаимодействие ИИ-ассистентов с конфигурациями 1С через протокол MCP.

2.2. Лицензия предоставляется на один компьютер (идентифицируется по Machine ID).

2.3. Передача кода активации или доступа третьим лицам не допускается.

3. Тарифы и оплата

3.1. Стоимость лицензии определяется выбранным тарифом и сроком. Актуальные тарифы размещены на сайте.

3.2. Доступны три уровня тарифа:

  • Аналитик — инструменты анализа кода и метаданных (только чтение)
  • Разработчик — дополнительно: редактирование кода, валидация
  • Архитектор — полный набор: редактирование метаданных, форм, отчётов

3.3. Оплата производится единовременно за выбранный период. Лицензия не продлевается автоматически.

3.4. После успешной оплаты Пользователь получает одноразовый код активации на указанный email.

4. Технические требования — важно прочитать до оплаты

⚠️ Пожалуйста, ознакомьтесь с этим разделом до оплаты.

4.1. Обязательные компоненты

Для работы ПО необходимо наличие:

  • 1C:EDT версии 2025.2.1 или новее
  • Установленного ИИ-ассистента, поддерживающего протокол MCP (Claude Code, Cursor, Continue, Cline и аналоги)
  • Открытого проекта конфигурации 1С в EDT

4.2. Рекомендуемый ИИ-ассистент

Разработка и тестирование выполнялись с Claude Code (Anthropic, подписка Pro $100/мес или выше). Другие MCP-совместимые ассистенты также работают, но могут потребовать более внимательного контроля.

4.3. Ограничение Cursor

⚠️ Важно для пользователей Cursor и тарифа «Архитектор».
Cursor поддерживает не более 40 инструментов по всем MCP-серверам суммарно. Тариф «Архитектор» включает 38 инструментов — при наличии других MCP-серверов часть инструментов может оказаться недоступна. Тариф «Разработчик» (26 инструментов) работает в Cursor без ограничений.

4.4. Требования к компьютеру

  • ОС: Windows 10/11 (64-bit)
  • RAM: от 16 ГБ, рекомендуется 32 ГБ (требование 1C:EDT, не плагина)

5. Активация и привязка к компьютеру

5.1. Код активации одноразовый и привязывается к компьютеру (Machine ID) при первом использовании.

5.2. При смене оборудования или переустановке ОС Machine ID может измениться — обратитесь в техподдержку.

5.3. В течение срока подписки предоставляются две бесплатные переактивации по уважительной причине.

6. Ограничения использования

Пользователь не вправе:

  • Передавать код активации или доступ третьим лицам

  • Декомпилировать или модифицировать ПО
  • Использовать ПО в целях, противоречащих законодательству РФ
